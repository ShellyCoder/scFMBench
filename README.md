# scFMBench
Systematic benchmarking of zero-shot utility and robustness in single-cell foundation models
This repository provides the code, evaluation metrics, and benchmarking pipeline for assessing single-cell foundation models (scFMs). The framework focuses on a zero-shot evaluation approach across multiple biological downstream tasks to test intrinsic representative power and structural robustness.

## Repository Structure

The project is organized into task-specific modules. Each directory contains the necessary evaluation logic, Python classes, and Jupyter Notebook demonstrations.

### Directory Overview

* **`example_code/`**: Contains the core evaluator scripts (e.g., `CellClusteringEvaluator.py`) and standard demonstration notebooks (e.g., `demo_cellclustering.ipynb`).
* 
* **`for_cell_clustering/`**: Example data for unsupervised grouping of cells.
* **`for_cell_annotation/`**: Example data for automated cell type identity assignment.
* **`for_drug_response/`**: Example data for predicting cellular perturbations and therapeutic sensitivities.
* **`for_batch_integration/`**: Example data for assessment of technical noise removal.
* **`for_gene_function/`**: Example data for evaluation of the model's ability to capture gene semantics.
* **`for_gene_regulation/`**: Example data for inferring gene regulatory networks (GRN).

## Getting Started

### Prerequisites

Ensure you have the following dependencies installed:
* Python 3.9.12
* Scanpy / AnnData
* PyTorch or TensorFlow
* NumPy, Pandas, Scikit-learn, scIB, RDKit

### Usage

The `example_code` directory is the primary entry point. The provided `.ipynb` files demonstrate the workflow:

1.  Load the pre-trained scFM and the input `.h5ad` data.
2.  Generate cell or gene embeddings.
3.  Initialize the task-specific `Evaluator` class.
4.  Execute the benchmark and output the metrics.

**Basic execution example:**

# 2.Cell Classification Evaluation (`for_cell_annotation`)

## Core Components

The evaluation strictly focuses on multi-class classification performance using macro-averaged metrics:
* **Macro-F1 Score**
* **Macro-AUROC** (Area Under the Receiver Operating Characteristic Curve)
* **Macro-AUPRC** (Area Under the Precision-Recall Curve)


# 3.Drug Response Prediction Evaluation (`for_drug_response`)

This module evaluates the capability of predicting drug sentivity. It employs a dual-tower neural network to integrate cell embeddings with drug molecular features.

## Core Components
* **Preprocessing Utilities:** Includes tools for converting drug SMILES strings into Morgan fingerprints using RDKit and standardizing response labels (e.g., sensitive vs. resistant).
* `DualTowerDrugResponse`: A PyTorch-based deep learning architecture that separately encodes cell and drug features before fusing them for classification.
* `DrugResponseEvaluator`: The primary evaluation class that manages data loading, model training, and performance validation across different experimental setups.

## Calculated Metrics
The evaluation focuses on binary classification performance (response vs. non-response) using the following metrics:
* **Macro-F1 Score**
* **AUROC** (Area Under the Receiver Operating Characteristic Curve)
* **AUPRC** (Area Under the Precision-Recall Curve)

## Quick Start

Ensure that the `rdkit` library is installed prior to running this evaluation.

```python
import os
import torch
import pandas as pd
from DrugResponseEvaluator import DrugResponseEvaluator

if __name__ == "__main__":
    # Set all paths to the example folder at the same directory level
    example_dir = "../example_data/for_drug_response/"
    drug_info_file = os.path.join(example_dir, "drug_response_14_datasets.csv")
    
    dataset_name = "GSE104987_MCF7"
    method_name = "Example_of_Method"  
    
    device = torch.device("cuda:0" if torch.cuda.is_available() else "cpu")
    print(f"Starting drug response prediction evaluation using device: {device}")

    # Initialize the evaluator
    evaluator = DrugResponseEvaluator(
        base_dir=example_dir,      
        embed_dir=example_dir,     
        drug_info_path=drug_info_file,
        device=device,
        epochs=20,                 # Use a smaller number of epochs for demonstration/testing
        lr=1e-3,
        patience=5,
        batch_size=64
    )

    # Run 5-fold cross-validation
    result_df = evaluator.run_5_fold(dataset_name=dataset_name, method_name=method_name)
    
    print("\n✅ Five-fold cross-validation test completed successfully! Results:")
    print(result_df)
    
    # Compute the mean of each evaluation metric
    summary = result_df[["MacroF1", "AUROC", "AUPRC"]].mean()
    print("\n📊 Overall average performance:")
    print(summary)
```

**1. Biological Conservation**

# 5.Gene Function Prediction Evaluation (`for_gene_function`)

This module evaluates the capacity of single-cell models to capture meaningful gene semantics. Specifically, it tests whether the generated gene embeddings can accurately predict tissue-specific gene functions using Human Protein Atlas labels.

## Core Components
* **`MLP`**: A PyTorch-based Multi-Layer Perceptron used as the downstream classification head to map gene embeddings to specific tissue labels.
* **`GeneFunctionEvaluator`**: The main evaluation class that handles embedding loading (from `.pkl`), label alignment, and orchestration of the training loop.
* **Label Alignment**: The `_prepare_labels` method automatically filters and aligns genes present in both the pre-trained embeddings and the ground-truth tissue database.

## Calculated Metrics
The script employs a 5-fold Stratified Cross-Validation strategy, computing the following multi-class classification metrics:
* **Macro-F1 Score**
* **Macro-AUC** (Area Under the ROC Curve)
* **Macro-AUPR** (Area Under the Precision-Recall Curve)

## Quick Start

```python
import os
import torch
import pandas as pd
from GeneFunctionEvaluator import GeneFunctionEvaluator

if __name__ == "__main__":
    # Paths pointing to the directory
    data_dir = "../example_data/for_gene_function/"
    emb_file = os.path.join(data_dir, "aligned_embeddings.pkl")
    tissue_file = os.path.join(data_dir, "HPA_tissue_specific_genes.pkl")

    # 1. Instantiate the evaluator
    evaluator = GeneFunctionEvaluator(
        emb_file_path=emb_file,
        tissue_genes_path=tissue_file
    )

    # 2. Specifically extract GeneCompass data for testing
    target_model = "GeneCompass"
    
    # Execute 5-fold calculation and return a DataFrame
    df_results = evaluator.run_5_fold(model_name=target_model)

    print("\n========== 📊 Final Results ==========")
    print(df_results)
```

# 6.Gene Regulatory Network Inference Evaluation (`for_gene_regulation`)

This module assesses the capability of single-cell models to infer Gene Regulatory Networks under transcription factor perturbations. It evaluates whether the models can identify true downstream target genes based on embedding differences between perturbed (Case) and unperturbed (Control) states.

## Core Logic
The evaluation computes the representation delta for each gene:
$\Delta(TF \rightarrow g) = |Embedding_{Case} - Embedding_{Control}|$

Genes are ranked by this absolute delta, and the ranked list is compared against experimentally validated ground-truth TF targets.

## Calculated Metrics
The script computes both global and threshold-specific metrics to evaluate target identification accuracy:
* **AUPRC** (Area Under the Precision-Recall Curve)
* **F1@5%** (F1 score for the top 5% ranked genes)
* **Jaccard@5%** (Jaccard similarity index for the top 5% ranked genes)

## Quick Start

```python
import os
import torch
import pandas as pd
from GeneRegulationEvaluator import evaluate_grn_performance

if __name__ == "__main__":
    TARGET_METHOD = "Cosine"
    TARGET_DATASET = "CDX1_OE"
    EXAMPLE_INPUT_DIR = f"../example_data/for_gene_regulation/" 
    EXAMPLE_META_DIR = f"../example_data/for_gene_regulation/" 

    print("="*50)
    print(f"Starting inferring the method: {TARGET_METHOD} | dataset: {TARGET_DATASET}")
    print("="*50)

    try:
        # Execute Single Dataset Evaluation
        df_perf = evaluate_grn_performance(
            method=TARGET_METHOD,
            dataset=TARGET_DATASET,
            input_dir=EXAMPLE_INPUT_DIR,
            meta_info_dir=EXAMPLE_META_DIR
        )
        
        print("\n✅ evaluation has been completed. The results for each indicator are as follows:")

```
